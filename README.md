# VS Code Projects

Collection of personal development projects and utilities.

## Projects

### PDF to Text Converter
Automatically converts PDF files to plain text format.

**Quick Start:**
```bash
python pdf_to_text_converter.py
```

See [notes/PDF_Converter_Setup.md](notes/PDF_Converter_Setup.md) for detailed documentation.

## Requirements
- Python 3.12+
- PyPDF2

## Installation
```bash
pip install PyPDF2
```
